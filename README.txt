Comment installer le jeu ?


> Vous pouvez compiler le jeu à l'aide d'un script jarify de votre choix.

Si tout s'est bien passé, vous devriez à présent voir un fichier game.jar dans le dossier.

Rien ne vous empêche également d'utiliser un IDE de votre choix.


Comment utiliser le jeu ?


> Vous pouvez utiliser la commande : java -jar <nom du fichier jar>

> Vous disposez également des script run, utilisez celui adapté à votre système d'exploitation.


Une fois lancé, merci de suivre les indications données par notre programme.
