Comment installer le jeu ?


> En supposant que vous utilisez une machine Linux, vous pouvez compiler le jeu en un executable grâce au script jarify présent dans le dossier.
Pour l’exécuter, lancez un terminal dans ce dossier puis tapez la commande suivante : ./jarify.sh

Si tout s'est bien passé, vous devriez à présent voir un fichier executable.jar dans le dossier.

Rien ne vous empêche également d'utiliser un IDE de votre choix.


> En supposant que vous utilisez une machine Windows ou Mac, vous devrez utiliser un IDE comme Netbeans, Eclipse ou encore IDEA pour importer notre code. Nous vous recommandons de le compiler en jar pour une meilleure expérience.


Comment utiliser le jeu ?


> Si vous avez choisi de créer le fichier jar, sur un Linux vous pouvez soit l'exécuter directement si vous savez déjà comment faire, soit utiliser le script run de la même façon que jarify. Sur une machine Windows ou Mac, vous devrez ouvrir un terminal dans le dossier où le jar se situe et réaliser la commande : java -jar <nom du fichier jar>

> Dans le cas de l'utilisation d'un IDE, nous vous invitons à vous renseigner sur comment faire selon ce que vous utilisez.


Une fois lancé, merci de suivre les indications données par notre programme.
